Project Description
 കൂടെ App is a Flutter-based mobile application that assists users in managing medicines through scanning, reminders, and voice-based alerts to improve medication adherence and safety.
The Problem Statement
Many users, especially elderly and busy individuals, forget to take medicines on time or misuse them due to unclear instructions, leading to health risks.

The Solution
Orma App solves this by scanning medicine details, providing timely reminders, and giving clear voice-based guidance to ensure correct and timely medication intake.
